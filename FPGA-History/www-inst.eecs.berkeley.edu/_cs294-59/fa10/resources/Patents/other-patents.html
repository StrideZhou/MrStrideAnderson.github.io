Altera, Actel, Lattice Patent Information

Compiled by John Lazzaro
This link:

http://www.google.com/patents?q=inassignee%3AAltera%20inassignee%3ACorporation&btnG=Search%20Patents&scoring=2&as_psrg=1

Should returned an ordered list of Altera issued patents, from the first one onward. If it doesn't work, then go to this web page:

http://www.google.com/patents

Type this in as the search:

inassignee:Altera inassignee:Corporation

and use the options on the left hand column to do: Sort by date (old first).

Likewise for Actel,

http://www.google.com/patents?tbs=bks%3A1&tbo=1&q=inassignee%3AActel%20inassignee%3ACorporation&btnG=Search%20Patents&scoring=2&as_psrg=1

Or else do this search manually:

inassignee:Actel inassignee:Corporation

If you look on the first page for the patents with Abbas El Gamal as first inventor, you'll find the ones that cover the famous fuse-programming algorithm.

Finally, Lattice's early patents included some important ones too:

http://www.google.com/patents?tbs=bks%3A1&tbo=1&q=inassignee%3ALattice%20inassignee%3ASemiconductor%20inassignee%3ACorporation&btnG=Search%20Patents&scoring=2

Or manually:

inassignee:Lattice inassignee:Semiconductor inassignee:Corporation
